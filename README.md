### 2020.6.1送给妹妹的节日礼物——出题器（持续更新）

#### 需要jre环境（我自己未精简）

**功能**

1. 加、减、乘、除选择
2. 变量最大数量选择
3. 是否存在小数，并控制小数位数
4. 题目数量
5. 控制数字的上下限
